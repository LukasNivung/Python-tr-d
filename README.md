# Python-tr-d
Umeå är som bekant 'björkarnas stad', och Umeå Kommun har följaktligen många björkar att sköta. För att hålla ordning har de en databas av alla träd som sköts av kommunen på opendata.umea.se. Det är mycket data och detta program i Python reder ut vad som finns och inte!

Det räknar antalet björkar i filen med olika regler (en simplare matchning, samt en mer generös överskattning), filtrerar fram björkar i närheten av en given punkt i staden, och introducerar därefter en objektorienterad struktur där varje träd representeras som ett objekt. Med hjälp av detta beräknas avstånd mellan träd och en referenspunkt.
